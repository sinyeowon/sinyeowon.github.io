## 풀이

```python
def solution(n):
    a, b = 0, 1
    
    for _ in range(n):
        a, b = b, a + b
    
    return a % 1234567
```

F(2) = F(0) + F(1) = 0 + 1 = 1

F(3) = F(1) + F(2) = 1 + 1 = 2

F(4) = F(2) + F(3) = 1 + 2 = 3

F(5) = F(3) + F(4) = 2 + 3 = 5

- 위와 같은 식으로 더해지기 때문에 `a, b = b, a+b`로 표현할 수 있었다.

- 그리고 문제에서 n번째 피보나치수를 1234567로 나눈 나머지를 구하라고 하고 있기 때문에 return 값을 `a%1234567`로 설정하였다.

```python
def solution(n):
    answer = 0
    
    answer = pibo(n)
    
    return answer

def pibo(k):
    if k == 0:
        return 0
    elif k == 1:
        return 1
    else:
        return pibo(k-1) + pibo(k-2)
```
