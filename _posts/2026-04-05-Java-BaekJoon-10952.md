[BaekJoon 10952](https://www.acmicpc.net/problem/10952)

### 풀이
``` java
import java.util.Scanner;

public class Main {
    public static void main(String args[]) {
        Scanner sc = new Scanner(System.in);
        
        while(true) {
            int a = sc.nextInt();
            int b = sc.nextInt();
            
            if((a==0) && (b==0)) {
                break;
            } else {
                System.out.printf("%d\n", a+b);
            }
        }
    }
}
```
Java에서는 보통 Python처럼 `while(1)`이 아니라 `while(true)`를 사용한다.
- Java는 조건식이 반드시 `boolean` 타입이어야 함
- Java에서 1은 `int`이므로 `boolean`으로 자동 변환이 안 됨

- Python에서는 `0(False), 1(True)`로 암묵적으로 처리됨
